Roomory Support

Roomory is a local mixed reality memory room experience for Meta Quest.

For support, questions, privacy requests, or store-review follow-up, contact daz_andy@outlook.com.

Basic Troubleshooting

  • Make sure your Meta Quest headset is updated and connected to Wi-Fi before installing Roomory from the Meta Horizon Store.
  • If Roomory asks for scene or room access, allow the permission so the app can read your Quest room setup.
  • If room data is missing, open Quest Space Setup, scan or select the current room, then return to Roomory and check again.
  • If imported photos, videos, or audio do not appear, try selecting the media again with the headset's system file picker.
  • If you want to remove local Roomory data, use the app's reset or clear-layout features where available, or uninstall Roomory from the headset.

Privacy

Roomory's first release is designed to run locally on the headset without a Roomory account, cloud sync, ads, analytics, subscriptions, or in-app purchases.

Read the Roomory Privacy Policy for details.

Roomory is published by DAZ, a personal independent creator brand. Support replies may take a few business days.
